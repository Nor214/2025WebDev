Web Technologies

HTML

An HTML, an abbreviation for Hypertext Markup Language, is the standard language used for web pages.
It is more specifically a coding language, in which websites use the code to translate into visuals that we can see and access.
So in general, it is the fundamental aspect for the creation of a web page or website.

CSS

CSS, standing for Cascading Style sheets, is a coding language that allows developers to control the visual aspects in their website.
This includes things such as elements, colors, fonts, text alightment, etc. It is also rather flexible, being able to be applied to various media types.

JavaScript

JavaScript is another coding language, this one being of more higher level. And it's primarily known as a language for web pages or sites.
It even integrates HTML and CSS, and it has a stretch of flexible variables. Overall, it is a higher level language mainly used for websites, and its Client-Side Scripting.
